Auto Recognition version 1.0 01.05.2022

Для запуска проекта:
1) Скопируйте проект и разархивируйте
2) Откройте консоль python 
3) Создайте виртуальную среду командой virtualenv auto_plate
4) Активируйте виртуальную среду командой source auto_plate/bin/activate
5) Установите зависимости из файла requirements.txt командой pip install -r requirements.txt
6) Запустите проект с помощью команды python manage.py runserver
7) Перейдите по ссылке, что возникнет в консоли
8) Добавьте в конец url /auto_plate
Теперь перед Вами открыта главная страница сайта. Можете загрузить картинку или видео и запустить распознавание.
